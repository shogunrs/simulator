Limitations of Traditional Approaches

01 Reactive Delays

Reacts after problems occur. 5–10 min delay causes SLA violations.

02 No Anticipation

Cannot learn temporal patterns or predict future workload.

03 Manual Tuning

Requires extensive manual configuration. Prone to thrashing.

04 Single Metric

Relies on CPU alone. Ignores memory, I/O, queue depth.

MDP Formulation

Markov Decision Process Framework

State (18D)

  • Performance: Latency, Rates
  • Resources: CPU, RAM (AWS/GCP)
  • Temporal: Hour, Day, Trend

Action (7)

  • 1. No-op
  • 2–3. Scale AWS (+/−)
  • 4–5. Scale GCP (+/−)
  • 6–7. Scale Both (+/−)

Reward

R = -10*SLA
    - 1*Cost
    - 0.5*Stability
07 / 16

PPO Algorithm

Proximal Policy Optimization

An Actor-Critic method selected for stable training.

  • Clipped Objective: Prevents destructive updates.
  • Sample Efficiency: Reuses data multiple times.
  • Stack: Gymnasium + Stable-Baselines3 + PyTorch.
ACTOR (Policy)
Maps State → Action
CRITIC (Value)
Estimates State Quality

Interpretation

Statistical Significance: The p-value < 0.001 indicates that the performance improvement is statistically significant with>99.9% confidence, ruling out random chance.

Practical Significance: Cohen's d = 3.76 represents a very large effect size (d > 0.8 is considered large), confirming that the improvement is not only statistically significant but also practically meaningful.

Validation: Results validated against AWARE: Automate Workload Autoscaling with Reinforcement Learning (Qiu et al., USENIX ATC 2023), a peer-reviewed academic benchmark demonstrating similar RL-based auto-scaling improvements in production cloud environments.

Note on Data: This study uses synthetic workload traces generated through simulation to enable controlled, reproducible experimentation. Workload patterns reflect realistic characteristics (temporal patterns, traffic spikes) based on publicly available traces and literature. ROI parameters use industry-representative benchmarks (Gainsight 2023, Pacific Crest SaaS Survey 2023).
